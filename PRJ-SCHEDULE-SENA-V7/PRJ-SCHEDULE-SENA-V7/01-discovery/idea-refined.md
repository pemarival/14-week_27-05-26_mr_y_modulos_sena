# Idea Refined

## Problem
SENA instructors struggle to manage schedules efficiently across multiple classrooms and times.

## Target User
SENA Instructors (full-time and contractors), Administrators.

## Value Proposition
A centralized, conflict-free scheduling system providing clear weekly views.

## MVP Scope
CRUD for Instructors, Classrooms, TimeSlots. Schedule grid with conflict detection. Observations on schedule entries.
