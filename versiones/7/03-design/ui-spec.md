# UI Spec

## Components
- **ScheduleGrid**: Matrix, rows=TimeSlots, cols=Classrooms. Green=Free, Blue=Booked.
- **InstructorList**: Table view with simple pagination or scrolling.
- **ConflictAlert**: Red toast notification top-right.

## Colors
- Primary: #004080 (SENA Blue equivalent)
- Success: #28a745
- Error: #dc3545
- Background: #f8f9fa
